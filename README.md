# Storybooks-Web-Application

This app lets the user login in with Google account, create, edit, delete stories. Set the stories as public and private. Look at all the stories of a specific user.
### Requirements 

- Node JS
- Express JS
- MongoDB
- Mongoose
- Passport
- Google OAuth
